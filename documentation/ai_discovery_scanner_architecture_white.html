🎤 Demo Script
"The scanner uses a modular discovery architecture. Each scanner module independently detects AI-related artifacts — model files, frameworks, agent code, runtimes, and API integrations. Findings are aggregated by the Scan Controller, classified into AI categories, and presented through reports and dashboards. This design allows us to add new detectors without changing the overall architecture."
